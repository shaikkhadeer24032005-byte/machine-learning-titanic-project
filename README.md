## Project Workflow

### Step 1: Data Loading and Preprocessing

* Loaded the dataset using Pandas
* Explored dataset structure and summary statistics
* Checked for missing values
* Handled missing data appropriately
* Encoded categorical variables
* Split dataset into training and testing sets (80:20)

### Step 2: Feature Engineering

* Analyzed feature importance
* Performed correlation analysis
* Selected relevant features for model training
* Removed unnecessary or less useful features

### Step 3: Model Training

The following Machine Learning models were trained:

1. Logistic Regression / Linear Regression
2. Random Forest
3. K-Nearest Neighbors (KNN) / Decision Tree

### Step 4: Model Evaluation

Models were evaluated using:

#### Classification Metrics

* Accuracy
* Precision
* Recall
* F1 Score

#### Regression Metrics

* RMSE
* MAE
* R² Score

A comparison table was created to compare model performance.

### Step 5: Best Model Analysis

* Identified the best-performing model
* Generated a Confusion Matrix (Classification)
  OR
* Generated a Residual Plot (Regression)
* Analyzed model performance and strengths
